22 /*
23  This code exists to allow data to go across the serial port as JSON-formatted
24  data. Doing so allows for different types of data (frames, debug messages,
25  configuration commands) to use the same UART simultaneously, while keeping
26  all data being conveyed as at least somewhat-readable using just a serial
27  terminal application.
28 
29  To facilitate readability within a terminal application, any binary data is
30  encoded as Base64.
31 */
